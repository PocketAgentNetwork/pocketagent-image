HOW TO BUILD & RUN:
===================

1. BUILD THE IMAGE
   cd "Product/PocketAgent Image/Cloud"
   docker compose build

2. CONFIGURE
   cp .env.example .env
   # Edit .env and add:
   # - OPENCLAW_GATEWAY_TOKEN (generate with: openssl rand -hex 32)
   # - Your LLM API keys (OPENAI_API_KEY, ANTHROPIC_API_KEY, etc.)

3. RUN
   docker compose up -d

4. ACCESS
   Open http://localhost:18789
   Complete onboarding (fill in IDENTITY, USER, JOB)
   Start chatting with your PocketAgent!

PERSISTENT STORAGE:
===================
The image uses Docker volumes for persistence:
- pocketagent-home:/home/node (agent's home directory)
- ./data/openclaw:/home/node/.openclaw (OpenClaw data)
- ./workspace:/home/node/.openclaw/workspace (agent workspace - seeded on first run)
- ./data/files:/home/node/files (agent-created files)

On first run, entrypoint.sh seeds the workspace from /app/workspace_init/
On subsequent runs, it uses the persistent workspace volume.
